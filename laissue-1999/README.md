# Laissue 1999 antennal-lobe atlas

Historical 3D reconstruction reference for the adult Drosophila antennal lobe.

Local-only source:

- `data/source/jcompneurol_1999_laissue_antlobe.pdf` is ignored by git.
- Source link is tracked in `../paper_pdf_sources.csv`.

Primary citation:

Laissue PP, Reiter C, Hiesinger PR, Halter S, Fischbach K-F, Stocker RF.
Three-dimensional reconstruction of the antennal lobe in Drosophila
melanogaster. Journal of Comparative Neurology. 1999;405:543-552.

The paper references FlyBrain accession `AB00200` for a VRML model. The original
VRML file has not been located yet.
