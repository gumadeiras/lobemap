# DoOR odor-response resources

Compact exports from the DoOR R packages for odor-response and glomerulus
metadata.

Run the 2D reference map:

```bash
./lobemap --atlas door
```

The napari view includes a glomerulus checklist and simple 2D map transforms
for inspecting the published map in mirrored or rotated orientations.

Generated by:

```bash
Rscript scripts/export_door_resources.R
```

Files in `data/source/`:

- `door_response_matrix.csv.gz`: normalized odor responses.
- `door_response_matrix_non_normalized.csv.gz`: non-normalized responses.
- `door_mappings.csv`: receptor, sensillum, OSN, glomerulus, and co-receptor
  mapping.
- `door_data_format.csv`: odor identifiers and metadata.
- `door_dataset_info.csv`: source dataset metadata.
- `door_response_range.csv`: per-study response ranges.
- `door_glo_dist.csv`: DoOR glomerulus distance matrix.
- `door_al_map_glomeruli.csv`: 2D glomerulus outline coordinates from DoOR.
- `door_al_map_labels.csv`: 2D glomerulus label positions from DoOR.
- `door_export_versions.txt`: package versions used for export.

Primary source:

Munch D, Galizia CG. DoOR 2.0: Comprehensive mapping of Drosophila melanogaster
odorant responses. Scientific Reports. 2016;6:21841.
doi:10.1038/srep21841.
