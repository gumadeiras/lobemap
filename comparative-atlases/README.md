# comparative antennal-lobe atlases

Reference-only comparative antennal-lobe papers and pages.

Downloaded source pages:

- `data/source/drosophila_suzukii_melanogaster_al_2025.html`
- `data/source/bactrocera_dorsalis_al_2018.html`

Paper PDFs are local-only and ignored by git. Source links are tracked in
`../paper_pdf_sources.csv`.

These are useful for species comparison and literature context. No reusable
3D source volumes or glomerulus meshes were found in the public article pages.
