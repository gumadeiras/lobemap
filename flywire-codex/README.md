# FlyWire / Codex connectome resources

Lightweight resources for the public FlyWire v783 connectome release.

Downloaded source files:

- `data/source/zenodo_10676866_metadata.json`: Zenodo record metadata.
- `data/source/zenodo_10676866_files.csv`: file manifest with sizes and download
  URLs.
- `data/source/proofread_root_ids_783.npy`: proofread root IDs from the release.

Large release tables are not included by default:

- `flywire_synapses_783.feather`: 9.5 GB.
- `proofread_connections_783.feather`: 852 MB.
- `per_neuron_neuropil_count_post_783.feather`: 234 MB.
- `per_neuron_neuropil_count_pre_783.feather`: 17 MB.

Primary source:

Dorkenwald S, et al. Neuronal wiring diagram of an adult brain. Nature. 2024.
FlyWire public release v783.
