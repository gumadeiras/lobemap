# Edmond FIB-SEM glomerulus datasets

Focused FIB-SEM resources for DA2 and DL5 glomerulus reconstruction datasets.

Source dataset:

https://edmond.mpg.de/dataset.xhtml?persistentId=doi:10.17617/3.WL9UQN

Downloaded lightweight metadata:

- `data/source/edmond_dataset_metadata.json`
- `data/source/VII_DA2_rAl.yaml`
- `data/source/X_DA2_rAl.yaml`
- `data/source/X_DL5_lAl.yaml`

Large archive and JSON files are not included by default. The Edmond metadata
records the file IDs and sizes so they can be downloaded when needed.
